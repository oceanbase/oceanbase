DBA_RSRC_PLAN_DIRECTIVES 
=============================================



功能 
-----------------------

`DBA_RSRC_PLAN_DIRECTIVES` 视图用于记录数据库中每组资源计划指令的详细信息。

相关视图 
-------------------------

* `DBA_RSRC_PLANS`

  

* `V$RSRC_PLAN`
