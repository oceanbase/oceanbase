V$PLAN_CACHE_PLAN_STAT 
===========================================



**功能** 
---------------------------

该视图记录了当前租户在当前 Server 上的计划缓存中缓存的每一个缓存对象的状态。
**注意**



该表不仅缓存了 SQL 计划对象，也缓存了 PL 对象，例如匿名块、PL Package 以及 PL Function 等，某些字段仅在特定对象下有效。
