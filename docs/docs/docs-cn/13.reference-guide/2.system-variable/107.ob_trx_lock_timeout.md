ob_trx_lock_timeout 
========================================

ob_trx_lock_timeout 用于设置事务的等锁超时时长，单位为微秒。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                        |
| 默认值     | -1                                                                                                         |
| 取值范围    | 有效取值范围：\[0,+∞)                                                                                             |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |
