system_memory 
==================================

`system_memory` 用于设置系统预留的内存容量。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 容量单位      |
| 默认值              | 30G       |
| 取值范围             | \[0M, +∞) |
| 是否重启 OBServer 生效 | 否         |


**注意**



系统预留内存不能分配给普通租户使用。
