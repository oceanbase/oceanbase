character_set_filesystem 
=============================================

character_set_filesystem 用于设置文件系统字符集类型。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | string                                                                                                     |
| 默认值    | BINARY                                                                                                     |
| 取值范围   | BINARY                                                                                                     |
| 生效范围   | * GLOBAL   * SESSION    |
| 是否可以为空 | 是                                                                                                          |
