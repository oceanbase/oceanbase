unit_balance_resource_weight 
=================================================

unit_balance_resource_weight 用于设置 UNIT 均衡策略中使用的资源权重，一般不需要手工配置。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |


**注意**



* 默认值为空。 如果设置，所有权重的总和必须为100。

  

* 当打开 enable_unit_balance_resource_weight 时， unit_balance_resource_weight 配置才能生效。
